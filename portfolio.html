This app uses HTML, CSS, JAVASCRIPT and JQUERY

This project was designed with the desire to inform and share fun facts about the city of chicago. It is meant to let those who call themselves "true Chicagoans" to back that statement up, and it also provides an environment for someone who doesn't know much about the city to learn some new and interesting facts.

While fully functional now and a clear demonstration of my skills as a developer, my hope is to continue to develop this web app and create an education resource for schools to use when teaching students about Chicago.
